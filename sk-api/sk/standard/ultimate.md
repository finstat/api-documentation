# Ultimat API
API slúži na vyžiadanie úplného detaily firmy

## Požiadavka ultimate
Výpis detailu firmy na základe parametra *ico* [`UltimateResult`](#UltimateResult)

> **Dopytovaná URL**: ```https://www.finstat.sk/api/ultimate```<br />
> **Hash parameter**: {ico}
